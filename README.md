## Fresh Installation

1. Install [bob](https://github.com/MordechaiHadad/bob)
2. Install nvim
2. Install tmux
3. Install TPM

```console
$ git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
